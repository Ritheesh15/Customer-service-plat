# **FOR CLIENT** 
cd client

npm install

or

yarn install

# **FOR SERVER**
cd client 

npm install

or

yarn install


# **RUNNING THE PROJECT**
**FOR CLIENT**

cd client
npm start
or
yarn start
The client will run on http://localhost:3000.

# **RUNNING THE SERVER**
**FOR SERVER**
cd ../server
npm start
or
yarn start
